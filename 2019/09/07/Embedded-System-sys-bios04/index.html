SYS/BIOS内核

SYS/BIOS并不是我们所认识的传统意义的操作系统，它只是完成一些最基础的一个系统所应包含的功能。所以一般情况下，称SYS/BIOS为一个内核更为合适。TI的一些文档里面也称SYS/BIOS为TI-RTOS Kernel.
